OptionsRisk Calculator

Smarter position sizing for options traders â€“ prevent account blowups, instantly.

Project Overview

OptionsRisk Calculator is a one-page web application that helps retail options traders calculate the optimal number of contracts to buy based on their account size, risk tolerance, and trade setup. Built with simplicity in mind, the tool empowers traders to apply professional risk management principles without needing a finance degree or spreadsheet.

Key Features

  • Position Sizing Calculator: Calculates how many option contracts to buy based on risk per trade, ensuring users never risk more than a set percentage of their account (3%â€“15%).
  • Risk-to-Reward Analysis: Instantly visualizes trade setups with clear color-coded R:R ratios and potential profit/loss calculations.
  • Smart Risk Guidance: Distinguishes between actual risk and capital committed to help traders understand both downside and exposure.
  • Live Price Fetching: Uses Finnhub API to fetch real-time stock prices and calculate strike distance from the current underlying.
  • Optional Advanced Inputs: Traders can optionally enter Delta, Theta, and IV to receive deeper insights without overwhelming beginner users.

Technical Stack

  • Frontend: React + TypeScript
  • Styling: Tailwind CSS
  • API: Finnhub (stock price data)
  • Hosting: Vercel

Why I Built This

Most retail traders fail because of poor risk management. This tool exists to make proper sizing, discipline, and R:R evaluation instant, visual, and foolproof â€” shifting traders from gambling to calculated execution.

Launch Date: April 11, 2025
